# Meet Your Vocaloid Waifu
당신의 최애 보컬로이드와 손가락 하트로 사랑을 나눠보세요 <3

### 주요기능
**당신의 Waifu를 선택**  
카사네 테토와 하츠네미쿠중 당신이 좋아하는 보컬로이드를 골라보세요

**웹캡에 사랑스럽게 등장하는 Waifu**  
당신이 선택한 보컬로이드가 웹캠에 나타나서 손하트를 날립니다.
같이 하트를 만들어 주세요 

**SSD 깊숙히 Waifu를 저장**  
스페이스바를 눌러 그녀를 당신의 맥북안에 영원히 저장하세요

**아쉽지만 이제는 만날 수 없는 그대**  
언젠가 이별의 때는 찾아옵니다.
아쉽지만 ESC를 눌러 다음 만남을 기약하세요
